# Credit Card Statement Parser

Convert Indian credit card PDF statements into normalized CSV transactions.

## Supported Cards

- YES Bank ACE RuPay

## Features

- PDF upload
- CSV export
- Local browser processing
- No server required

## Privacy

All processing happens locally in your browser.
