# Credit Card Statement Parser

Convert credit card PDF statements into CSV files.

## Supported Cards

- YES BANK KreditPe ACE
- Jupiter CSB edge+

## Features

- Browser-only processing
- No server
- No data upload
- Editable transactions
- CSV export
